## 🎯 المميزات الرئيسية

### 1️⃣ **تحميل التقرير**

```html
📁 Input Field: أدخل رقم الملف
🔵 Button: تحميل التقرير
```

**الخطوات:**
1. أدخل معرف الملف (مثلاً: 45)
2. انقر على "تحميل التقرير"
3. انتظر جزء من الثانية لتحميل البيانات

**النتيجة:**
- ✅ تحميل جميع الإحصائيات
- ✅ عرض المجالات
- ✅ عرض التقييمات التفصيلية

---

### 2️⃣ **الإحصائيات الفورية**

عند تحميل التقرير، سترى 4 بطاقات:

```
┌─────────────┐  ┌─────────────┐  ┌─────────────┐  ┌─────────────┐
│   🟢 Strong  │  │  🟡 Emerging │  │  🔴 Limited  │  │  ⚪ Insuff.  │
│   Count: 2   │  │  Count: 1    │  │  Count: 0    │  │  Count: 0   │
│   67%        │  │  33%         │  │  0%          │  │  0%         │
└─────────────┘  └─────────────┘  └─────────────┘  └─────────────┘
```

**كل بطاقة تعرض:**
- 🎨 **لون مميز** حسب الحالة
- 📊 **عدد التقييمات** في هذه الحالة
- 📈 **النسبة المئوية** من الإجمالي

---

### 3️⃣ **تقسيم المجالات**

```
📚 Domain Cards Grid
├── التخطيط والإعداد (D1)
│   └── عدد الأدلة: 2
├── تنفيذ الدرس (D2)
│   └── عدد الأدلة: 1
└── ...
```

**معلومات كل بطاقة:**
- 🏷️ اسم المجال (بالعربية)
- 🔢 كود المجال (D1, D2, etc.)
- 📋 عدد الأدلة المجمعة

---

### 4️⃣ **جدول التقييمات التفصيلية**

```
┌──┬────────┬────────────────────────┬──────────┬────────────┬────────────┬──────────┐
│# │ KPI   │ KPI Name               │ Status   │ Evidence   │ Domain     │ Date     │
├──┼────────┼────────────────────────┼──────────┼────────────┼────────────┼──────────┤
│1 │ 1.1   │ وضوح الأهداف          │ Strong   │ [Strong]   │ التخطيط    │ 2026-03  │
│2 │ 1.2   │ المحتوى العلمي         │ Emerging │ [Emerging] │ التخطيط    │ 2026-03  │
│3 │ 2.3   │ تفاعل الطلاب          │ Emerging │ [Emerging] │ التنفيذ    │ 2026-03  │
└──┴────────┴────────────────────────┴──────────┴────────────┴────────────┴──────────┘
```

**الأعمدة:**
- **#**: رقم الـ row
- **KPI**: رمز المعيار (1.1, 2.3, etc.)
- **KPI Name**: اسم المعيار كامل
- **Status**: الحالة (Strong/Emerging/Limited/Insufficient)
- **Evidence**: النص الدليل
- **Domain**: اسم المجال
- **Date**: تاريخ التقييم

## 🔍 ميزات البحث والفلترة

### البحث بالنص 🔎

```
Search Box: بحث في النصوص...
```

**الاستخدام:**
```
- اكتب كلمة أو جزء من النص
- النتائج تحدّث تلقائياً
- دعم البحث بالعربية والإنجليزية
```

**أمثلة:**
- "وضوح" → البحث عن كل التقييمات التي تحتوي على "وضوح"
- "1.1" → البحث عن المعيار رقم 1.1 فقط
- "Strong" → البحث عن التقييمات التي توصفها بـ Strong

### تصفية حسب الحالة 📊

```
Status Filter: [All] [Strong] [Emerging] [Limited] [Insufficient]
```

**الاستخدام:**
```
- انقر على الحالة المطلوبة
- يتم إعادة تصفية الجدول فوراً
- اختر "All" لإظهار الكل
```

### البحث المدمج 🎯

**مثال متقدم:**
```
1. اكتب "تفاعل" في Search
2. انقر على "Strong" في الفلتر
3. النتيجة: جميع التقييمات التي تحتوي على "تفاعل" وحالتها Strong
```

## 📄 PAGINATION (التنقل بين الصفحات)

```
[السابق] [الصفحة 1 من 5] [التالي]
```

**المعلومات:**
- **عدد العناصر:** 10 عناصر لكل صفحة
- **الملاحة:** أزرار السابق والتالي
- **الحالة:** تعطيل الأزرار في البداية والنهاية

**الخطوات:**
```
1. انقر "التالي" للانتقال للصفحة التالية
2. انقر "السابق" للعودة
3. اعرض معلومات الصفحة الحالية أعلى الجدول
```

## 📥 تصدير البيانات

### زر التصدير

```
📥 تصدير JSON
```

**ما يتم تنزيله:**
- ملف JSON يحتوي على:
  - بيانات الملف الصوتي
  - جميع التقييمات
  - الإحصائيات الكاملة
  - معلومات التقرير

**اسم الملف:**
```
evaluation_45_2026-03-09.json
```

**محتوى الملف:**
```json
{
  "metadata": {
    "fileId": 45,
    "exportDate": "2026-03-09T20:33:25Z"
  },
  "evaluations": [...],
  "statistics": {
    "strong": 2,
    "emerging": 1,
    "limited": 0,
    "insufficient": 0
  }
}
```

## ⚠️ رسائل الخطأ

### عدم العثور على الملف

```
❌ خطأ: الملف غير موجود أو لا توجد تقييمات
```

**الحل:**
- تحقق من رقم الملف
- تأكد من تقييم الملف بالفعل

### خطأ في الاتصال

```
❌ خطأ: فشل في اتصالال بالخادم
```

**الحل:**
- تأكد من تشغيل الخادم: `npm run dev` في backend
- تحقق من الـ URL
- أعد تحميل الصفحة

### البيانات فارغة

```
ℹ️ لا توجد تقييمات لهذا الملف
```

**الحل:**
- تأكد من وجود تقييمات فعلاً
- جرب ملف آخر

## ⚙️ تروبل شوتينج

### المشكلة: Dashboard لا يفتح
**الحل:**
```
1. تأكد من: npm run dev في frontend يعمل
2. الـ URL صحيح: http://localhost:5173/evaluations
3. أعد تحميل: F5 أو Ctrl+R
```

### المشكلة: "خطأ في الاتصال"
**الحل:**
```
1. تأكد من: npm run dev في backend يعمل
2. الـ IP صحيح: localhost:5000
3. أعد تشغيل: الخادم
```

### المشكلة: الجدول فارغ
**الحل:**
```
1. تأكد من: رقم الملف صحيح
2. الملف: تم تقييمه فعلاً
3. جرّب: ملف آخر معروف
```

### المشكلة: التصدير لا يعمل
**الحل:**
```
1. تأكد من: متصفح يدعم download
2. الـ Path: صحيح (/:id/evaluation/export)
3. تحقق من: Console للأخطاء
```
